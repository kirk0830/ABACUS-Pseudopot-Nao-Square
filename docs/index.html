Welcome

Welcome to ABACUS-Pseudopotential-Numerical atomic orbital Square (APNS). APNS aims to provide pseudopotentials and numerical atomic orbitals with high efficiency and precision for ABACUS user simulating. APNS is mainly developed and maintained by ABACUS-AISI team.

Pseudopotential test

This section provides pseudopotential test for basis_type lcao calculation. In this test suites, not only norm-conserving but also ultrasoft pseudopotentials are tested for each element as long as availble. For basis_type pw calculation referring pseudopotential and plane wave combinations, please refer to Pseudopot-Pw test section instead. In this section ecutwfc is tested for cohesive energies, pressure and zon-boundary phonon frequencies, converged ecutwfc is provided for user.
